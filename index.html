University of Washington Campus Map

An interactive UW Campus map built with React, TypeScript, and Leaflet. Utilizes the directed graph ADT to create the map. The buildings themselves were provided by a dataset including a list of long names and short names for each building. The appliation utilizes Dijkstra's shortest path algorithm to find the shortest path from one building to another.
